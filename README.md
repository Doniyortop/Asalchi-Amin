# Asalchi Amin - Магазин натурального мёда

Интернет-магазин натурального мёда и продуктов пчеловодства из Самарканда.

## Новые возможности (v1.1)
- 🚀 **База данных SQLite**: Теперь ваши заказы, пользователи и товары надежно сохраняются в БД.
- 🔍 **Поиск по каталогу**: Удобный поиск мёда по названию и описанию.
- 👤 **Личный кабинет**: Пользователи могут видеть историю своих заказов и их статус.
- ✨ **Улучшенный UI**: Плавные анимации, современный поиск и отзывчивый дизайн.
- 🛠️ **Админ-панель**: Полное управление товарами, заказами и модерация отзывов.

## Технологии
- Node.js + Express.js
- EJS (шаблонизатор)
- SQLite3 (база данных)
- Leaflet (интерактивные карты)
- CSS3 (анимации и переменные)

## Запуск локально
1. `npm install`
2. `node app.js`

## Деплой на Render
Сайт настроен для работы на Render.com. 

### ⚠️ Важно: Сохранение данных
На бесплатном тарифе Render файлы (включая `database.sqlite` и `sessions.sqlite`) удаляются при каждом перезапуске или деплое. Чтобы ваши товары, заказы и **авторизация пользователей** не пропадали:

1.  В панели управления Render перейдите в раздел **Disks**.
2.  Создайте новый диск (например, на 1 ГБ) и примонтируйте его по пути `/opt/render/project/src/data`.
3.  В настройках приложения измените пути к БД на этот диск (например, через переменные окружения).

*Без подключения диска данные будут сбрасываться после каждого обновления сервера.*
