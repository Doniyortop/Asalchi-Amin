# Asalchi Amin - Магазин натурального мёда

## Описание
Интернет-магазин натурального мёда и продуктов пчеловодства из Самарканда.

## Технологии
- Node.js + Express.js
- EJS шаблонизатор
- SQLite/In-memory база данных
- Multer для загрузки файлов
- Bootstrap CSS
- Leaflet для карт

## Запуск локально
```bash
npm install
node app.js
```

## Функции
- 🍯 Каталог товаров
- 🛒 Корзина покупок
- 👤 Регистрация и авторизация
- 📦 Оформление заказов
- 🗺️ Выбор точки доставки на карте
- 👨‍💼 Админ-панель управления товарами и заказами
- 🌐 Мультиязычность (RU, UZ, EN)
- 📱 Мобильная адаптация
- 🔍 SEO оптимизация
